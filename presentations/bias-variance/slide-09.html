§ 2 — Bias & Variance › Variance

Variance

Var[f̂(x)]  =  𝔼D[(f̂D(x) − 𝔼D[f̂D(x)])²]    # This is just 𝔼[(· − its own mean)²].
True process
Y = f(x) + ε,  𝔼[ε|X] = 0
#f(x) = the true value we want to predict
#ε — irreducible error; no model can predict it.
One training set
D = {(x₁,y₁), …, (xₙ,yₙ)}
Fitted model
f̂D(x) — depends on D
#Each draw of D gives a different fitted model
Expectation
𝔼D[f̂(x)] = avg prediction across all possible D
#Averages over the randomness of the training set itself (imagine rerunning the world)
What it means
If you retrained on a different sample, how much would predictions move? Variance measures that scatter.
What causes it
Model too sensitive to its training set — high-degree polynomials, unpruned trees, k=1 nearest neighbour.
What doesn't cause it
Variance is about instability across datasets, not being wrong on average.
