Serial vs Parallel Layers
Serial (standard)
Input x
↓
Attention
↓
FFN
↓
Output
Attention runs first, its output feeds into FFN. Sequential dependency.
Parallel (faster)
Input x
↓ split
Attention
FFN
↓ sum
Output
Both run simultaneously on the same input. Reduces latency on multi-GPU setups.
Parallel layers trade some model quality for wall-clock speed — the two sublayers no longer condition on each other, but the savings in communication overhead can be significant at scale.
