Stability Tricks
01
Softmax Division by Zero
Softmax in attention and output can produce very large logits → numerical instability. Goal: keep the denominator close to 1.
02
Z-Loss (Auxiliary Loss)
Lz = ε · log²(Z)
Penalizes large partition function Z in softmax. Keeps logits small, denominator near 1, training stable.
03
QK LayerNorm
Apply LayerNorm to Query and Key vectors before computing attention softmax. Directly prevents exploding attention logits.
04
Logit Soft-Capping
logits = c · tanh(logits / c)
Clamps logits to range [−c, +c] via smooth tanh. Prevents extreme values without hard clipping.
Rule of thumb
Add LayerNorms when training becomes unstable. Each norm adds minimal overhead but significantly smooths the loss landscape.
Why denominator ≈ 1?
When the softmax partition function Z is close to 1, the distribution is well-normalized and gradients flow cleanly. Large Z → saturation → vanishing gradients.
