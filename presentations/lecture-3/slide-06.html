Key Hyperparameters
FFN Dimension
d_ff = 4 × d_model
Standard rule. With GLU activations, scale down by 2/3:
d_ff = 8/3 × d_model
to keep parameter count comparable.
Exception: T5 uses d_ff = 64 × d_model.
Attention Heads
head_dim × num_heads = d_model
More heads → smaller head_dim, proportionally.
But head_dim × num_heads can exceed d_model — output projection handles the mismatch.
Depth vs Width (Aspect Ratio)
aspect ratio = d_model / n_layers
Width can be parallelised across GPUs (layer sliced into matrices).
Depth cannot — layers must run sequentially.
Trade-off: wide & shallow vs narrow & deep.
Vocabulary Size
30–50k  /  100–250k
Single language: ~30–50k tokens.
Multilingual models: ~100–250k tokens to cover scripts and languages.
